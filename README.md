# Lex Iuridicus Site

Sitio web construido desde cero con **Next.js (JavaScript)** y **Tailwind CSS**.

## Stack
- Next.js 14
- React 18
- Tailwind CSS 3

## Comandos
```bash
npm install
npm run dev
npm run lint
npm run build
```

## Objetivo de esta base
- Tener una landing moderna para servicios jurídicos empresariales.
- Dejar una base escalable para seguir avanzando en JavaScript con componentes interactivos.
